## 🎯 한 줄 요약
"원고 쓰는 작가"에서 "출판 플랫폼을 소유한 1인 출판사 대표"로 넘어가기 위한 **A부터 Z까지 실무 매뉴얼**. 사업자등록부터 ISBN 발급, 인쇄·물류 계약, 서점 입점, 일일 운영(SCM)까지 한 번에 훑는다.

---

## 1. 왜 1인 출판사를 소유해야 하는가

| 이유 | 설명 |
|------|------|
| ⚡ **스피드** | 출판사 투고 후 선정 대기로 몇 달~몇 년 잡아먹는 동안 콘텐츠 신선도 사망. 직접 소유하면 원천 차단. |
| 💰 **이윤 극대화** | 일반 작가 인세는 보통 **10%**. 자가 출판은 인쇄·택배·서점 할인율을 직접 통제해 **30~50%** 가능. |
| 📦 **시작 조건** | 콘텐츠가 **5종 이상**이면 규모의 경제 작동. 1~2종부터는 단가 압박이 큼. |

---

## 2. 출판사 설립 — 4단계

1. **사업자등록** — 일반과세자로. 출판/방송통신/온라인정보제공업 코드 (인강 때문에 면세 X). 출판 자체는 면세지만 이런 부수업까지 다루려면 일반과세 선택.
2. **출판사 로고 제작** — 외주 또는 자체. 반드시 `AI / PNG / JPG` 3종 모두 확보. AI는 일러스트레이터 원본이라 수정 가능.
3. **출판사 신고 확인증** — 관할 **구청 문화체육관광과**에 신고. 택배 수령 시 분실 위험, 직접 수령 권장.
4. **국립중앙도서관 가입** → **서지정보유통지원시스템**에서 ISBN 발급. 신청 시 임시 번호가 먼저 발급돼 원고 작업 중 사용 가능.

> 💡 **출판사명 ≠ 사업자등록증의 회사명.** 브랜드명을 따로 써도 무방.

## 3. 책 제작 — 원고에서 인쇄까지

### 원고 & 편집
- 본문 작성: 한글(HWP) 또는 인디자인. 단행본은 보통 인디자인.
- **외주 비용 기준** (200페이지 단행본 기준):
  - 본문 인디자인 편집: 페이지당 약 **3,000원** → 약 60만 원
  - 교정·교열·윤문: 페이지당 **1,000~1,500원** (퇴직 국어교사 등 풀이 큼)
- ✅ 교정·교열은 **반드시 외주.** 본인 글의 오타·어색함은 본인 눈에 안 보임.

### 사진·그림·폰트 (저작권 지뢰)
- 사진·그림은 **반드시 직접 촬영/제작.** 다른 곳에서 따오면 큰일.
- 폰트는 **상업용 라이선스** 필수. 윤디자인 멤버십 약 88,000원/년 추천. 무료 대안: **코펍월드체**(한국출판인회의 배포, 단 재판매 X).

### 인쇄용 PDF (초보 함정 1번)
> ❌ 한글 메뉴의 "PDF로 저장하기"는 **인쇄용으로 못 씀.** 품질 저하 + 글꼴 미포함.
>
> ✅ "인쇄 → 프린터를 PDF 드라이버로 선택" → 인쇄소가 요구하는 사양(고품질, 출판품질, 글꼴 포함 등)으로 설정해서 출력.

## 4. 외부 파트너 세팅 (4종)

| 구분 | 역할 | 결정 기준 |
|------|------|-----------|
| **지업사** | 종이 공급 (백상지/미색모조/크래프트 등) | 단가 + 종이 수급 안정성 + 규모. 책 성격 말하면 어떤 종이 쓰라고 알려줌 |
| **인쇄소** | CTP 인쇄 (옛 옵셋 대체, 품질 거의 동등) | 규모 (CTP 기계 1대 7~8억) + 빠른 공급 (200p × 3,000부 ≈ 48시간) |
| **제본소** | 본드로 묶어 책 형태 완성 | 보통 인쇄소가 제본소와 협업 관계라 인쇄소만 잡으면 같이 진행 |
| **물류창고** | 보관 + 서점 배송 + 반품 처리 | **안정성**(화재보험 잘 든 곳, 누수 없는 곳). 너무 싼 곳 위험. 한번 정하면 바꾸기 매우 번거로움 |

**카페 "[꿈꾸는 책 공장](https://cafe.naver.com/madebook)"** 에서 견적 요청 글 올리면 사장님들이 견적 보내줌. (인쇄소, 물류, 디자이너 등)

**물류 비용 예시:** 3,000부 보관 시 월 **50~60만 원** (택배비 포함). 1,000부 정도면 약 20만 원 후반.
**물류업체:** 날개(업계 1위), 협진, 해피데이(파주, 1인 출판사 다수 사용) 등.

> ⚠️ **집에서 책 1,000권 보관·발송은 사실상 불가.** 부피·습도·시간 모두 안 됨.

## 5. 인쇄 → 입고 → 발주

- 발주 시 **여분**으로 50~100부 더 찍히는 게 보통. 샘플·여유분으로 활용.
- 책은 "**한 덩이**" 단위(약 150×850×850, 초등학생 키 정도)로 묶여 팰릿 + 지게차 + 화물차로 물류창고 입고.
- 입고 후 물류창고 시스템에 부수가 잡히면 서점 발주 가능 상태.

---

## 6. 서점 계약 — 6대 메이저

대상: **교보문고, 예스24, 알라딘, 영풍문고, 인터파크도서, 북센(또는 동등 도매)**

### 공급률 (서점이 출판사에서 책을 가져갈 때 지불 비율)
- 표준: 정가의 **60~65%** (예: 2만 원 책 → 출판사가 1.2만~1.3만 원 수취)
- **70%까지 협상 가능 조건:** 업력 3년 이상 + 누적 매출 3억 이상 + 서점 매출 기여 인정
- 정산은 **다음 달**. 단, 안 팔리면 **반품**으로 돌아옴.

### 계약 방식 (위탁 vs 매절)
| 방식 | 특징 | 적합 |
|------|------|------|
| **위탁** | 안 팔리면 반품 가능. 출판사가 재고 리스크 | 수요가 불확실한 책 |
| **매절** | 서점이 매입 (반품 X). 보통 50부 이상 발주 시 매절 조건 | 수요 확실 OR 현금 회전 급함 |

### 교보문고는 별종
- **우편 계약 X, 직접 방문 필수.**
- 책 **2권** 들고 와야 함 (다른 서점은 1권).

## 7. 일일 운영 — SCM 시스템

계약하면 각 서점의 SCM(Supply Chain Management) 시스템 계정 부여.

| 서점 | 알림 방식 |
|------|-----------|
| 교보문고 | 아침 8:10경 문자/메일 |
| 예스24 | 문자 알림 |
| 인터파크 | 문자 알림 |
| 알라딘 | 알림 없음 — 직접 들어가서 확인 |

매일 아침 발주 확인 → SCM에서 출고 처리 → 물류창고가 배송.

## 8. 출간 마케팅

1. **신간 등록** — 각 서점 신규 등록 양식 (분야별 MD 담당자가 안내). 메일로 등록 양식 보내면 인터넷 서점에 노출.
2. **온라인 채널** — 블로그, 카페, 유튜브, 페이스북, 인스타그램에 빠짐없이 정보 배포.
3. **스마트스토어** — ISBN 책도 **면세상품**으로 등록해 판매 가능. 세금신고 시 면세/과세 분리.

## 9. Q&A 정리

| 질문 | 답 |
|------|-----|
| ISBN 도서 개인이 스마트스토어 판매? | ✅ 가능. 면세상품으로 등록. |
| 최소 인쇄 부수? | 옛 옵셋 3,000부 → CTP 인쇄로 **500부**도 가능 |
| 200p 책 인쇄비? | 종이 약 150만 원 + 인쇄·제본 ≈ 총 300만 원대 |
| 표지 디자인 도구? | **일러스트레이터 권장.** 캔버는 중복·완성도 이슈로 비추 |
| 세네카(책등) 두께 계산 | 종이 두께 × 페이지 수 ÷ 2 + 표지 두께. 코팅지는 약 0.2mm 가산 |
| 폰트 비용 | 유료 권장. 윤디자인 멤버십 약 88,000원/년 (공제 가능) 또는 무료 코펍월드체 |

## 🧭 첫 1인 출판사 만들기 — 한눈에 보는 순서

```
[설립]
사업자등록 → 로고 → 출판사 신고 → 국립중앙도서관 가입 → ISBN 발급
   │
[제작]
원고 → 디자인(외주) → 교정·교열·윤문(외주) → 인쇄용 PDF → 사진/그림/폰트 확보
   │
[파트너]
지업사 + 인쇄소 + 제본소 + 물류창고 계약 ("꿈꾸는 책 공장" 활용)
   │
[입고]
인쇄 → 한 덩이 단위 포장 → 화물차 → 물류창고 입고 → SCM 부수 잡힘
   │
[유통]
6대 서점 계약 (위탁/매절) → SCM 가입 → 매일 아침 발주 처리
   │
[마케팅]
신간 등록 → 블로그/카페/유튜브/SNS → 스마트스토어 추가 판매
```

## 🪞 핵심 인사이트 5선

1. **"플랫폼 소유"가 핵심 — 인세 10% vs 자가 30~50%의 격차.** 수익보다 더 큰 건 콘텐츠 신선도(스피드) 통제권.
2. **5종 이상 콘텐츠 확보 후 진입.** 1~2종으로는 규모의 경제가 안 돌아간다.
3. **외주 줘야 할 것 / 내가 해야 할 것 명확.** 디자인·교정은 외주, 인쇄용 PDF 세팅·서점 계약·SCM 운영은 내가.
4. **물류창고 선택은 안정성.** 한번 정하면 책 다 끄집어내야 옮길 수 있어서 사실상 종신계약.
5. **교보는 별종 처리.** 다른 서점과 같은 룰로 접근하면 시간 낭비.
